Generate documentation

The documentation is automatically generated using doxygen. To (re-)generate the documentation, you must have doxygen installed on your machine. Asssuming you have unpacked the Artifact submission tarball and entered the top-level artifact directory, 

  1. cd cuba/doc
  2. make
To remove the documentation, use 
make clean
