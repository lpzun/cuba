1. Installation

Note: CUBA is being developed in C++11. Make sure that your compiler supports C++11 or above. CUBA has been successfully compiled with g++, clang and cygwin.

1.1 Install from source

The following assumes you have unpacked the artifact and entered the top-level artifact directory. 
make
cd bin
./cuba -h
After returning to the top-level artifact directory, you can use make clean to remove the object files. For a thorough cleanup, which also removes the executable, use make distclean.

1.2 Using the binaries

  • Unpack the CUBA binary for your architecture from the .zip file above. This will produce a single file, the executable for the platform you chose.
  • Run it.

The Tutorial tab contains more details about running the tool.
